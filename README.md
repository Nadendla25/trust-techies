About

EduNavigator is designed for efficient, self-paced learning. By offering structured topics across multiple subjects, it helps students focus on relevant materials based on their level:

Subjects Available:

Python

Artificial Intelligence (AI)

Machine Learning (ML)

Natural Language Processing (NLP)

Computer Vision (CV)

Levels: Beginner, Intermediate, Advanced

Each topic includes:

Notes – Online tutorials and articles

Videos – YouTube tutorials

PPTs – Slide decks for concept review

Practice Questions – Hands-on exercises and interview questions

🔹 Features

User Authentication: Register and login using email and password stored in browser localStorage

Dynamic Dashboard: Load topics based on selected subject and level

Curated Resources: Access notes, videos, PPTs, and questions for each topic

Responsive Design: Card-based UI with hover effects for better UX

Lightweight: No backend required, works entirely in the browser

🔹 Technologies

HTML5 – Structure and layout

CSS3 – Styling, card layouts, hover effects

JavaScript – Dynamic content loading, event handling, and localStorage authentication
